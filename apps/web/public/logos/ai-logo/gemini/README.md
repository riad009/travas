# Gemini AI Logo Batch

This folder contains:

- prompts.json — the 79 extracted Aqurion logo prompts
- individual response files from Gemini test runs
- generated images once a valid Gemini API key is used

Current status:

- The batch workflow is ready
- The placeholder key value "banana" was tested and rejected by the Gemini API as invalid
- The shared bulb reference image is included as reference-bulb.png one folder up

To generate the set with a real key, run the batch script from the project root and set your valid Gemini API key in the environment first.
